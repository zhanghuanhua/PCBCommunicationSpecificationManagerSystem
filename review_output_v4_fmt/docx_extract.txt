一、文档概述
本文档为铠铂 EAP 系统(EAP)与设备(以下简称 EQP)进行业务数据传输与回传接口文档 ， EAP 系统与 EQP 对接时需按照本文档的规范进行接口开发配置。
二、 约定
EAP 系统与设备相互集成接口统一采用 Restful WebAPI 接口方式，请求方式为 POST，请求和响应的主体是 JSON 结构，使用的编码是 UTF-8；具体定义如下所示：
[TABLE 47]
接口方式 | Restful WebAPI ，基于 Http 协议开发
请求方式 | POST
Content-Type | application/json
编码规则 | UTF-8
安全验证 | 安全验证
请求方 IP 地址 | 通过解析 HttpRequest 获取请求方 IP 地址，系统判断请求方 IP 地址是否在白名单
1.接口参数定义：
a)输入参数：即接口请求时输入参数，指接口调用时的输入参数，采用 JsonObject 格式，参数如果为主从表，则采用 Json 嵌套数组结构；
b)返回值参数：即接口调用返回值参数，指接口调用时的返回值，采用 JsonObject 格式；
2.基础访问地址：
[TABLE 52]
